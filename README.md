# 🌸 Perfumy – HTML & CSS Perfume Website

A responsive and visually elegant perfume brand website built during my self-learning journey. This project showcases my understanding of HTML5, CSS3, responsive web design, and modern frontend development concepts. Inspired by perfume branding websites, Perfumy focuses on clean layouts, attractive visuals, and a user-friendly browsing experience.

🔗 **Live Demo:** [https://perfumy-css-html.vercel.app/](https://perfumysite.vercel.app/)

## 📚 Learning Journey

This project was developed as part of my frontend development practice and self-learning process. I learned HTML and CSS concepts through YouTube tutorials and hands-on practice, particularly from the **Error Makes Clever** channel.

Through this project, I gained practical experience in building responsive web pages, designing attractive layouts, and applying modern CSS styling techniques.
