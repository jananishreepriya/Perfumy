🌸 Perfumy – HTML & CSS Perfume Website

A responsive and visually elegant perfume brand website built during my self-learning phase. This project demonstrates my ability to structure multi-section layouts, apply custom CSS styling, and deploy static sites using Vercel.

- Responsive design: Adapts well to different screen sizes  
- Clean navigation: Simple header links for smooth user flow  
- Contact section: Includes email and phone info with styled layout
